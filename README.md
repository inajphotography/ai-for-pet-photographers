## Setup: Get Ready in 5 Minutes

Follow these steps once before you start the course. You only do this once.

---

### Step 1 — Get a Claude Subscription

You need a **Claude Pro** ($20/month) or **Max** subscription.

Go to [claude.com/pricing](https://claude.com/pricing) and subscribe.

This gives you access to both Claude on the web AND Claude Code. No API keys needed.

---

### Step 2 — Open Your Terminal

**Mac:** Press `Cmd + Space`, type `Terminal`, hit Enter

**Windows:** Press `Win + R`, type `powershell`, hit Enter

If you've never opened Terminal before, that's completely fine. You're just going to type a few lines. That's it.

---

### Step 3 — Install Claude Code

Copy and paste this into your terminal, then press Enter:

**Mac / Linux:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://claude.ai/install.ps1 | iex
```

This installs Claude Code on your computer. Takes about 30 seconds.

---

### Step 4 — Sign In

Type this in your terminal and press Enter:

```bash
claude
```

A browser window will open. Sign in with your Claude account and authorise Claude Code.

Done. You're connected. No API keys, no tokens, no configuration.

---

### Step 5 — Create Your AI Workspace Folder

This is important. Claude Code reads your business brain file from wherever you are when you open it. This step makes sure it always finds yours.

**Mac / Linux:**
```bash
mkdir -p ~/Documents/my-ai-assistant && cd ~/Documents/my-ai-assistant
```

**Windows:**
```powershell
mkdir "$HOME\Documents\my-ai-assistant"; cd "$HOME\Documents\my-ai-assistant"
```

This creates a dedicated folder for your AI assistant files. **Every time you open Claude Code going forward, navigate here first.** Your CLAUDE.md, your research reports, and your client guide all live here.

---

### Step 6 — Download the Course

Paste this into your terminal and press Enter:

```bash
curl -sL https://raw.githubusercontent.com/inajphotography/ai-for-pet-photographers/main/install.sh | bash
```

This downloads all 4 lessons and your 4 gift files to your machine. Takes about 5 seconds.

---

### Step 7 — Start the Course

Type this in your terminal (from your my-ai-assistant folder) and press Enter:

```bash
claude
```

Then type:

```
/inaj:start
```

That's it. You're in.

---

## Every Time You Use Claude Code

1. Open Terminal
2. Type `cd ~/Documents/my-ai-assistant` and press Enter
3. Type `claude` and press Enter
4. You're back in your AI assistant

---

## Questions?

If something isn't working, the most common fix is making sure you're in your `my-ai-assistant` folder before opening Claude.

Built questions into the lessons — just tell Claude what's happening and it'll help you sort it out.
